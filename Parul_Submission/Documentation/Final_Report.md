# Bluestock Mutual Fund Capstone Report

## 1. Executive Summary
This capstone project develops a complete analytics workflow for Bluestock mutual fund data, covering data ingestion, database loading, exploratory analysis, performance evaluation, and dashboard preparation. The project demonstrates how structured financial data can be transformed into decision-ready insights for mutual fund performance monitoring and investor analysis.

## 2. Objectives
- Build a reliable ETL pipeline for mutual fund data
- Create a normalized SQLite database for analytics use
- Explore patterns in fund characteristics, NAV movement, SIP behavior, and investor transactions
- Prepare a Power BI dashboard structure for interactive reporting
- Deliver a submission-ready project package with clear documentation

## 3. Data Sources
The analysis uses the following datasets:
- Fund master data
- NAV history
- Monthly SIP inflows
- Scheme performance
- Investor transactions

## 4. Methodology
1. Data ingestion from CSV files into a consistent project workflow
2. Data cleaning and structural validation
3. Database creation and table loading in SQLite
4. Exploratory analysis and visualization
5. Performance metric preparation and dashboard design planning

## 5. Key Findings
- Equity-oriented schemes form the dominant fund categories in the available data.
- NAV behavior varies meaningfully across funds, indicating differentiation in risk-return profiles.
- SIP and transaction activity reveal investor participation patterns that can support segmentation strategies.
- The dashboard structure prepared for Power BI supports investment overview, fund analysis, and investor insights.

## 6. Analytical Components
- ETL and data quality checks
- Exploratory data analysis
- Performance metric preparation
- Investor and SIP analytics
- Interactive dashboard planning

## 7. Business Relevance
The project provides a practical framework for understanding mutual fund performance, monitoring investor engagement, and supporting portfolio and marketing decisions. The deliverables are suitable for academic submission and can be extended into a more advanced production workflow.

## 8. Conclusion
The Bluestock Mutual Fund Capstone project successfully integrates data engineering, analytics, and visualization into a cohesive submission package. It demonstrates a strong foundation for future work in dashboard automation, live NAV ingestion, and advanced portfolio analytics.
